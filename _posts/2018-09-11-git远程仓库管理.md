查看：
```
git remote -v
```

添加：
```
git remote add origin git@github.com:michaelliao/learngit.git
```
修改：
```
git remote set-url origin git@github.com:michaelliao/learngit.git
```
或者：
```
git remote rm origin
git remote add origin git@...git
```
